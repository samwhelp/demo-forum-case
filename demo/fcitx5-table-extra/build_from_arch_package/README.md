## Step


### help

run

``` sh
make
```

or run

``` sh
make help
```


### prepare

run

``` sh
make prepare
```


### download

run

``` sh
make download
```


### extract

run

``` sh
make extract
```


### build

run

``` sh
make build
```


### install

run

``` sh
make install
```


### fcitx5-install

run

``` sh
make fcitx5-install
```


### fcitx5-config

run

``` sh
make fcitx5-config
```


### use-fcitx5

run

``` sh
make use-fcitx5
```


### clean

run

``` sh
make clean
```
