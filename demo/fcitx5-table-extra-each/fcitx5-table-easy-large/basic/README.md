## Step


### help

run

``` sh
make
```

or run

``` sh
make help
```


### prepare

to run [prepare.sh](prepare.sh)

``` sh
make prepare
```


### download

to run [download.sh](download.sh)

``` sh
make download
```


### build

to run [build.sh](build.sh)

``` sh
make build
```


### install

to run [install.sh](install.sh)

``` sh
make install
```


### uninstall

to run [uninstall.sh](uninstall.sh)

``` sh
make uninstall
```


### clean

to run [clean.sh](clean.sh)

``` sh
make clean
```
