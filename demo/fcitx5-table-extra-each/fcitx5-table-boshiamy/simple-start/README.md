## Usage


### help

run

``` sh
make
```

or run

``` sh
make help
```


### install

to run [install.sh](install.sh)

``` sh
make install
```


### uninstall

to run [uninstall.sh](uninstall.sh)

``` sh
make uninstall
```


### clean

to run [clean.sh](clean.sh)

``` sh
make clean
```
